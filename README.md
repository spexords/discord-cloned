## Description
This full stack project was completely inspired by Discord application. It's written in ASP. NET Core 5.0 and React.
## Live demo
https://discordcloned.herokuapp.com/
## Backend
- ASP .NET Core 5.0
- Entity Framework Core
- SignalR
- Auto Mapper
## Frontend
- React
- Redux
- Axios
- Styled Components
- React Hook Form
